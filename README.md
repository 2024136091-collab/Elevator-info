# 승강기 정보검색 앱

> 공공데이터를 활용해 승강기 설치·점검·위치 정보를 한 곳에서 조회하는 Flutter 앱

## 승강기 정보란?

승강기(엘리베이터)는 건물 내 사람과 화물을 수직으로 이동시키는 설비입니다.
대한민국에는 약 **80만 대 이상**의 승강기가 설치·운행되고 있으며,
안전한 운행을 위해 정기적인 점검과 관리가 법적으로 의무화되어 있습니다.

### 주요 정보 항목

| 항목 | 설명 |
|---|---|
| 설치 정보 | 건물명, 주소, 설치 연도, 제조사, 최대 정원 |
| 점검 이력 | 점검 날짜, 점검 결과, 담당 검사원 |
| 위치 정보 | 지도상 설치 위치 (위도·경도) |
| 운행 상태 | 정상 운행 / 운행 정지 / 점검 중 |

### 데이터 출처

행정안전부 **승강기 안전관리 공공데이터 API** (data.go.kr) 를 활용합니다.

## 개요

| 항목 | 내용 |
|---|---|
| 플랫폼 | Flutter (Android / iOS) |
| 데이터 | 행정안전부 승강기 안전관리 공공 API |
| 상태 관리 | Riverpod |
| 로컬 저장 | Hive |

## 주요 기능

- **승강기 검색** — 건물명·주소·승강기 번호로 검색
- **점검 이력 조회** — 최근 점검 날짜·결과·담당자 확인
- **위치 조회** — 지도에서 설치 위치 확인
- **상세 정보** — 제조사·설치연도·최대 정원 표시
- **즐겨찾기** — 자주 사용하는 승강기 로컬 저장

## 화면 구성

| 화면 | 설명 |
|---|---|
| 홈 | 검색창, 최근 조회 목록 |
| 검색 결과 | 승강기 목록, 필터링 |
| 상세 정보 | 스펙, 점검 이력 |
| 지도 | 설치 위치 마커 표시 |
| 즐겨찾기 | 로컬 저장 목록 관리 |

## 기술 스택

- **Flutter** — 단일 코드베이스 Android·iOS 지원
- **Riverpod** — 상태 관리, 비동기 API 처리
- **Hive** — 즐겨찾기 오프라인 저장
- **Google Maps** — 위치 조회 지도 표시
- **Dio** — HTTP 클라이언트, 공공 API 연동
